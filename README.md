# Dungeon Vibes

A 3D roguelike dungeon exploration game with survival mechanics, inspired by Cult of the Lamb, built with Three.js.

## Overview

Dungeon Vibes is a browser-based 3D roguelike game where players explore procedurally generated dungeons, collect resources, fight enemies, and try to survive as long as possible. The game features:

- 3D graphics powered by Three.js
- Procedurally generated dungeons
- Survival mechanics (health, hunger, etc.)
- Combat system with various enemies
- Resource collection and management
- Character progression

## Development

This project is built using:

- Three.js for 3D rendering
- JavaScript/TypeScript for game logic
- Vite for fast development and building
- Node.js for package management

## Getting Started

### Prerequisites

- Node.js (v14+)
- npm or yarn

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/gillworks/dungeon-vibes.git
   cd dungeon-vibes
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. Start the development server:

   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Controls

- WASD: Move character
- Mouse: Look around
- Left Click: Attack
- E: Interact
- Space: Jump
- Shift: Sprint
